### Description
This is a real-time Rock Paper Scissors game built using Python and Computer Vision.
The game detects hand gestures using a webcam and plays against an AI opponent.

### Technologies Used
- Python
- OpenCV
- MediaPipe
- AI Logic

### Features
- Real-time hand gesture detection
- AI opponent with random move generation
- Live result display on screen

### How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the game:
   python main.py

### Learning Outcome
- Computer Vision basics
- AI logic implementation
- Real-time application development
